Skills

After taking a class in high school, practicing individually with python, and taking this class. I have become very proficient in front end web development using HTML, CSS, and JavaScript.

I have been programming in python for almost 5 years at this point. Creating projects such as calculators, algorithm testers, data visualizations, as well as basic websites and databases.

I have taken and completed Advance Placement Computer Science which revolves around Java programming. In it I got to learn a vast amount of what I know today of Object Oriented Programming. As well as complex algorithms and how to implement them.

I picked up graphic design using Affinity designer not only as a hobby but to further my abilities as a programmer and web designer. It is what allowed me to create my own hero image for this site. It is a skill that has come in handy much more than I would have guessed at first glance.
